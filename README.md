# Analyse de la proximité de la population avec les déchetteries

Ce projet s'inscrit dans le cadre du datalab du Parlement Ouvert.


## Outils utilisés

Outils principaux:
- OSRM
- PostgreSQL
- PostGIS

Convertion er préparation des données:
- csvkit (csvcut)
- GNU Parallel
- gdal/ogr (ogr2ogr)

Modules python:
- psycopg2
- requests
